Arch Linux Installation Commands
==========================================


Essentials
Mounting HDD
sudo mkdir /mnt/hdd
UUID=4ACAE882CAE86B9F /mnt/hdd  ntfs-3g  defaults  0  0


Packages
*IMP* Install yay (AUR helper)
sudo pacman -S --needed git base-devel
git clone https://aur.archlinux.org/yay.git
cd yay
makepkg -si

Packages from official repos (pacman)
sudo pacman -S --noconfirm discord mpv feh steam obs-studio neovim ntfs-3g qbittorrent

Packages from AUR (yay)
yay -S --noconfirm spotify zen-browser-bin faugus-launcher noctalia-shell
